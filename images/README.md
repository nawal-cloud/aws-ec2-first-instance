# AWS EC2 First Instance Project

## Overview
Launched first EC2 instance (Amazon Linux 2023) with Apache web server.

## Steps
1. Created AWS Free Tier account
2. Launched EC2 instance (t2.micro)
3. Configured Security Group (SSH + HTTP)
4. Connected via SSH and installed Apache
5. Verified Apache test page via public IP
